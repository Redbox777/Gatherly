# Gatherly

Telegram-бот для организации встреч, путешествий и совместных мероприятий.

## Возможности

- Встречи — создание, приглашение участников, RSVP
- Места — сохранение интересных локаций
- Чек-листы — списки дел для встреч
- Погода — текущая погода в любом городе
- Восход/закат — время рассвета и заката
- Голосования — опросы внутри встреч
- Расходы — учёт трат и расчёт баланса
- Маршруты — точки маршрута для поездки
- Ссылки — общие ссылки внутри встречи
- Статистика — личная статистика пользователя
- Напоминания — автоматические уведомления о встречах

## Архитектура

Core + Telegram — бизнес-логика отделена от интерфейса.

core/ содержит чистые сервисы без Telegram-зависимостей.
telegram/ содержит только интерфейс, вызывает core.

## Запуск локально

pip install -r requirements.txt
cp .env.example .env
Впиши токен в .env
